Acerca de...

Mercado de Divisas es una página web creada en Angular y la API pública Frankfurter.

Se trata de un proyecto para el módulo Desarrollo web en entorno cliente del Grado Superior de Desarrolo de aplicaciones Web, impartido en BIRT.eus.

Creado por Sergio Moreno (smoreno@birt.eus) el 16 de mayo de 2024.

Otros recursos utilizados:

  • MockAPI.io: Para crear la API REST del libro de visitas.
  • Bootstrap: Framework para dar estilo a la página.
  • Moment.js: Librería JavaScript para la manipulación de las fechas.
  • Chart.js: Librería JavaScript para la creación de gráficos.
  • Flaticons.es: Descarga del logotipo.
